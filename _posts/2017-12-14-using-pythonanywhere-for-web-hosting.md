## Write a test script
```
$ mkdir ~/bin
$ mkdir ~/log
```
```bash
#!/bin/bash

# create filename
timestamp=`date +%Y%m%d%H%M%S`
fname=hello_$timestamp.log

# write message
echo "Message generated by hello script" >> $fname
echo `date` >> $fname

# move file to /log directory
mv $fname ~/log
```

## Set up `cron`
```
$ export $EDITOR=vi
$ crontab -e
01 * * * * uname /home/uname/bin/hello
```
